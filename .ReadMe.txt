要開始編輯專案 請在載入後開啟 ./Assets/Scenes 中的場景
./Build/ 存放遊戲執行檔
ctrl(hold) + `(down) 開關系統日誌(目前僅設定選單會log)
	 (tab鍵上方)

遊戲流程:
{
[堆塔模式]
階段開始後生成目標點(魚)及方塊
畫面右上角由上到下顯示:當前方塊堆高度, 目前已生成方塊數/目標方塊數, 限時倒數(僅貓咪模式)
在所有方塊落地 且玩家不再移動方塊 3秒後進入貓咪模式
[貓咪模式]
玩家須在時限內以鍵盤操控貓咪爬上方塊堆到達目標點
否則扣血並進入補救模式
到達目標點後貓咪退場並結束此階段
[補救模式]
生成一定數量方塊給玩家完善方塊堆
在所有方塊落地 且玩家不再移動方塊 3秒後進入貓咪模式

**所有模式中若方塊或貓咪落出平台將導致扣血, 貓咪將會重新入場**

重複以上模式三階段後玩家通關(畫面左下角魚影代表目前遊戲階段)
若在此期間血量歸零則遊戲結束
}
